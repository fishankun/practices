# Shapes in C

This folder contains simple C programs for printing different shapes and patterns using loops and conditional statements.

These programs are useful for practicing:

- Nested loops
- Pattern printing
- Logic building
- Character and number formatting
- Problem solving in C

## Included Shape Patterns

Some examples of shapes included in this folder:

- Triangle
- Pyramid
- Inverted Pyramid
- Diamond

## Example

Example of a pyramid pattern:

```text
    *
   ***
  *****
 *******
*********
```
## Technologies Used

Language: C
Compiler: GCC / Any standard C compiler

## How to Run

Compile a program:

```Bash
gcc filename.c -o output
```
Run the program:

```Bash
./output
```

## Purpose

This folder was created for practicing pattern-related problems and improving programming fundamentals.
