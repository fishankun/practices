# Number Guessing Game in C

A simple number guessing game written in C where the player tries to guess the total of two hidden numbers within 5 attempts.

## Features

- User gets 5 chances to guess the correct number
- Gives hints whether the guessed number is greater or smaller
- Uses loops and conditional statements
- Ends immediately when the correct answer is guessed

---

## Game Rules

- Two numbers are stored in the program
- Their total is calculated secretly
- The player must guess the total
- The game provides hints after each wrong guess
- The game ends after:
- guessing the correct number, or
- using all 5 attempts

---

## Technologies Used

- C Programming Language
- Standard Input/Output Library (`stdio.h`)
